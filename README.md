# weather-app

Sovelluksen käyttäjä voi:<br><br>

- hakea säätiedot kaupungin nimellä<br>
- nähdä nykyisen sään<br>
- nähdä esimerkiksi:<br>
  - lämpötilan<br>
  - tuulen nopeuden<br>
  - ilmankosteuden<br>
  - sääkuvauksen<br>
Sovellus hakee tiedot ulkoisesta sää-API:sta.<br><br>

Esimerkkejä rajapinnoista:<br>

- OpenWeatherMap API<br>
- WeatherAPI
